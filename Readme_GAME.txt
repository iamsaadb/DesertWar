***********DESERT WAR GAME ReadMe*************************
**********************************************************

This project was created using NetBeans IDE and Java 8.0 (Update 181). 

Directory: /DesertWar

How to run the projet on NetBeans IDE:
1- Click on File in IDE
2- Click "Open Project"
3- Browse and select DesertWar folder 
4- Click OK to Open it
5- Once open, run by clicking on Run button in the toolbar, or by right-clicking the 
class DesertWarGame.java in DesertWar Package



Key Controls:

********* Player 1  (Top left Tank) ***************
*						  *
*	A ---> Left Rotation			  *			  
*	W ---> Move Forward			  *
*	D ---> Right Rotation			  *	
*	S ---> Move Backwards			  *
*   SPACE ---> Shoot bullets			  *	
*						  *
*						  *	 
***************************************************

********* Player 2  (Bottom Right Tank) ***********
*						  *
*	LEFT ARROW KEY ---> Left Rotation	  *			  
*	  UP ARROW KEY ---> Move Forward          *
*      RIGHT ARROW KEY ---> Right Rotation	  *	
*	DOWN ARROW KEY ---> Move Backwards	  *
*                ENTER ---> Shoot bullets         *	
*						  *
*						  *	 
***************************************************
